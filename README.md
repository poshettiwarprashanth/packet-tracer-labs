# packet-tracer-labs

This repository documents my practical learning journey in networking and cybersecurity using Cisco Packet Tracer. It contains hands-on labs designed to build a strong foundation in network concepts while developing the skills required for cybersecurity roles.

The labs progress from beginner to advanced topics and focus on understanding how networks operate in real-world environments. Each exercise includes objectives, network topologies, device configurations, observations, troubleshooting steps, and key takeaways.

## Topics Covered

Topics Covered
* Network fundamentals
* OSI Model and TCP/IP Model
* IP addressing and subnetting
* MAC addressing and Ethernet fundamentals
* ARP (Address Resolution Protocol)
* Router and switch configuration
* VLANs and inter-VLAN communication
* Static and dynamic routing
* Default gateways and packet forwarding
* DHCP configuration and troubleshooting
* ICMP and network connectivity testing
* NAT concepts and implementation
* Access Control Lists (ACLs)
* Network troubleshooting methodologies
* Security-focused networking concepts
* Packet flow analysis and traffic inspection
* Real-world network design scenarios
* Cybersecurity-oriented networking practices

## Repository Goals

* Develop practical networking skills through hands-on labs.
* Strengthen the foundational knowledge required for cybersecurity careers.
* Build a well-documented portfolio demonstrating continuous learning and problem-solving abilities.
* Showcase the ability to design, configure, analyze, and troubleshoot network environments using industry-standard tools.

This repository reflects my transition from software development into cybersecurity and represents my commitment to continuous learning through practical experience and documentation.
