# Lab 01 - Cisco Packet Tracer Basics

## Objective

The objective of this lab is to become familiar with the Cisco Packet Tracer environment and understand basic networking devices, interfaces, cables, and topology creation.

## Devices Used

* 1 Cisco 2960 Switch
* 2 PCs

## Topology

PC0 ----- Switch ----- PC1

## Connections

| Device | Interface     | Connected To |
| ------ | ------------- | ------------ |
| PC0    | FastEthernet0 | Switch Fa0/1 |
| PC1    | FastEthernet0 | Switch Fa0/2 |

## Cable Used

Copper Straight-Through Cable

## Observations

* Link lights initially appeared amber/orange.
* After initialization, link lights turned green.
* Green status indicates that the physical connection is operational.

## Key Concepts Learned

* Packet Tracer Interface
* Network Devices
* End Devices
* Switch Functions
* Router Functions
* Ethernet Interfaces
* Copper Straight-Through Cable
* Real-Time Mode
* Simulation Mode

## Mini Exercise Answers

### Q1. Which cable did you use?

Copper Straight-Through Cable

### Q2. Which switch ports did you connect?

Fa0/1 and Fa0/2

### Q3. What color did the links become after initialization?

Green

### Q4. What is the purpose of Simulation Mode?

Simulation Mode allows users to observe packet movement step-by-step and analyze protocols such as ARP, ICMP, DHCP, and TCP within the network.

## Outcome

Successfully built and connected a basic LAN topology consisting of two PCs and one switch using Cisco Packet Tracer.
